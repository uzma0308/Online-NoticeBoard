Quick Guide
1. To add a notice, go to notice tab and write the notice to the given area and press done button.

2. To delete a notice, press on the delete icon press bwlow all the notices.

3. To edit notice,tap on edit icon and made changes and press done.

4. You can add category to the notice so that it will get filtered according to the category.

5. You can see your profile by going to profile tab.
